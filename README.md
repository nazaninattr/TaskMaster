# Task Manager

This is my CS50 Final Project.

A web-based task manager built with Flask and SQLite that allows users to manage tasks with priorities, deadlines, and categories.

## Features
- User authentication
- Create, edit, delete tasks
- Task priorities
- Deadlines
- Task completion
- Categories

## Tech Stack
- Flask
- SQLite
- HTML / CSS

## How to Run
```bash
flask run
